# benstroud's Homebrew Tap

Homebrew tap for shipping my Github hosted tools.

## Install

```bash
brew tap benstroud/tap
```

## Install Packages

```bash
# formula
brew install benstroud/tap/<name>

# cask
brew install --cask benstroud/tap/<name>
```
